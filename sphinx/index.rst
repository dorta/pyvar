Variscite Python API for Machine Learning
=========================================

|license|

.. |license| image:: https://img.shields.io/badge/License-BSD%203--Clause-blue

**Pyvarml** is a Python package that provides a set of high-level abstraction to
allow any user to develop their own Machine Learning application in a simplified
and efficiently way without writing too many lines of code.

* See a few examples written by using the Variscite Python API for Machine Learning:
